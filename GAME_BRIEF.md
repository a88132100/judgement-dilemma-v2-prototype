# Game Brief：審判困境 v2

## Game Name

審判困境 v2

## One-line Pitch

玩家在公開承諾與暗中選擇之間做出合作或背叛的心理博弈，透過發言、功能牌與裁決點數結算，在審判局的遊戲中爭取生存與勝利。

## Player Fantasy

玩家扮演被神秘組織「審判局」抓捕的成員。

每位玩家都被迫參與一場合作與背叛的審判遊戲。

玩家必須在公開場合說服別人相信自己，同時判斷其他人的真實意圖。

## Core Fun

核心樂趣是：

- 公開承諾與暗中選擇之間的心理壓力
- 相信或懷疑他人的判斷
- 守信與失信的風險選擇
- 功能牌造成的資訊戰與反制
- 每回合裁決點數變化帶來的壓迫感

## Core Loop

每回合流程：

1. 玩家公開承諾合作或背叛
2. 玩家進行發言
3. 玩家暗中選擇最終陣營
4. 玩家可選擇使用功能牌
5. 公開型功能牌依順序處理
6. 暗放型功能牌於揭示階段處理
7. 揭示最終陣營
8. 系統依局勢計算裁決點數
9. 玩家補牌
10. 進入下一回合

## Key Decisions

玩家每回合的重要決策：

1. 我要承諾合作還是背叛？
2. 我要守諾還是失信？
3. 我要相信誰的發言？
4. 我要使用功能牌還是保留手牌？
5. 我要保護自己，還是干擾別人？
6. 目前局勢適合合作還是背叛？

## Failure Risk

這個玩法最可能失敗的地方：

1. 規則過於複雜，新玩家難以上手。
2. 背叛策略如果太強，合作會失去意義。
3. 背叛策略如果太弱，心理博弈會不足。
4. 功能牌如果太複雜，會讓結算變得難以理解。
5. 玩家可能不清楚自己為什麼加分或扣分。
6. 遊戲可能變成隨機猜測，而不是有效判斷。

## Validation Goal

第一版 Prototype 要驗證：

1. 4 人局是否能形成足夠心理壓力。
2. 承諾與失信機制是否有趣。
3. 裁決點數結算是否平衡。
4. 功能牌是否提供有效反制。
5. 玩家是否願意玩第二局。
6. 是否存在單一策略過強的問題。
